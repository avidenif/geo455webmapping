How to use this map:

Map Interactivity
  • Home button: Click this button to go back to study area.
  • Thematic layers toggle: Click these layers on/off in the layer menu.
  • Basemap toggle: Switch between imagery and street basemaps.
  • Popups: Click on county/pin point/red polygon to view more information on the feature.
  • Search tool: Use this search tool to look up any state in the Southwestern region of the United States.
  • Disclaimer button: Click this button to view an important dislcaimer about the map.
  • Mini map: Use this minimap to view where you are in the United States.

Data Sources and Methodology

Data sources
  • Historical sites: 10 extraordinary Native American cultural sites protected on public lands
  • Reservation polygons (2022): US Domestic Sovereign Nations: Land Areas of Federally-Recognized Tribes (BIA)
  • Population Choropleth (2024): DPO5 ACS Demographic and Housing Estimates
Data processing
  1. Download shapefiles into ArcGIS Pro
  2. Simplify/clean data fields and/or polygons
  3. Project to proper projection
  4. Filter to study area if needed
  5. Add new fields and calculate fields if needed
  6. Join with other shapefiles if needed
  7. Export to GeoJSON
  8. Load into leaflet map in Pheonix Code using fetch
  9. Add/edit necessary CSS and features
  10. Push to GitHub
Web delivery methods
  1. GeoJSON
